CERTIFICATE FILES GUIDE
=======================

Place your certificate PDF files in this folder with the following names:

NPTEL Certificates:
- nptel-cloud-computing.pdf
- nptel-industry-4.0-iiot.pdf
- nptel-iot.pdf

Oracle Certificate:
- oracle-cloud-infrastructure.pdf

Cisco Certificate:
- cisco-ethical-hacking.pdf

Internship Certificate:
- internship-product-marketing.pdf

IMPORTANT:
- Use PDF format for best compatibility
- Make sure filenames match exactly (case-sensitive)
- You can also use images (.jpg, .png) but update the file extensions in index.html accordingly

The certificates will open in a new tab when clicked.
